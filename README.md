## Welcome to FlashcardJDA

This project has been created to help pupils of the [Jeanne d'Arc establishment](https://www.jda-argentat.fr/) to revise using flashcards.

We use the following liraries:
  * jQuery 3.4.1
  * Bootstrap v4.4.1
  * Popper.js
